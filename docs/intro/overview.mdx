# Overview

Lit is an internet native security and identity network.  With Lit, you can create and control keys and run code for virtually any application that can be made private, immutable, and / or autonomous.  

:::info

Lit is still in developer preview with the mainnet beta launching soon. Follow [@LitProtocol](https://twitter.com/litprotocol) on X for mainnet updates.

[Learn more](/v3/network/networks/testnet) about what data will persist during the developer preview.
:::

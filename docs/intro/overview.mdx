# Overview

Lit Protocol provides encryption, signing, and wallet infrastructure as-a-service to developers building in web3. Spanning across a decentralized set of nodes, the Lit Network provides a fault-tolerant and programmable alternative to centralized key management systems and custodians. 

:::note

Lit is still in developer preview with the mainnet launching soon. Follow [@LitProtocol](https://twitter.com/litprotocol) on X for mainnet updates.

[Learn more](https://developer.litprotocol.com/v3/network/state-of-network) about what data will persist during the developer preview.
:::
